פרויקט 6 - מזג אוויר

כדי ליצור את האפליקציה השתמשתי בתוכנות:
html, javascript, css with sass.
האפליקציה מביאה את כל הנתונים על עיר בעולם בעזרת openWeather api.
האפליקציה מאפשרת למשתמש/ת לקבל את:
פרטי מזג האוויר, התחזית השבועית והתחזית היומית.
עיצוב האפליקציה משתנה בהתאם לנתונים, כך חווית המשתמש/ת היא כל פעם שונה.
למשל העיצוב בעיר מישראל יהיה שונה מהעיצוב בעיר מארצות הברית.
ה - api של המזג האוויר הוא חצי חינמי ולא מביא את הנתונים המלאים ולכן:
התחזית היומית מוצגת בהפרש של כל 3 שעות ולא כל שעה.
התחזית השבועית מוצגת רק חמישה ימים קדימה ולא כל השבוע.
האפליקציה היא רספונסיבית.
