פרויקט 1 - פרויקט מדינות

כדי ליצור את האפליקציה השתמשתי בתוכנות:
javascript, html , css with bootstrap.
האפליקציה מביאה את כל המדינות בעולם בעזרת countries api של מדינות.
הפרטים שאפשר לראות הם, היבשת, המיקום ביבשת(מערב ומזרח וכו'), הדגלים, האוכלוסייה
השפות המדוברות, וסוג המטבעות
המשתמש/ת יכול/ה לשמור מדינות ולסמן אותם בלייק
ולהיכנס יום אחרי ובעזרת כפתור המדינות הפייבוריטיות להיזכר במדינות שנבחרו.
כמו כן ברגע שהמשתמש/ת מתחיל/ה לכתוב אפילו אות אחת,
האפליקציה תעבור סינון אוטומטי
וכל המדינות שיש בהן את אותה אות יופיעו על המסך.
כך ניתן לחפש למעשה כל מדינה בעולם מבלי לרדת ולחפש בכל האפליקציה.
האפליקציה היא רספונסיבית.
