פרויקט 2 - פרויקט עיצוב/בניית מסכים

כדי ליצור את האפליקציה התשמשתי בתוכנות:
javascript, html, css with sass.
האפליקציה נותנת למשתמש/ת לעצב עמוד בעזרת 5 אלמנטים שונים.
האלמנטים הם:
טבלה, פסקה, כותרת, רשימה וקופסה.
האפשרויות לעיצוב האלמנט יופיעו לאחר שהמשתמש/ת יבחר/תבחר איזה אלמנט הוא/היא רוצה ליצור.
הטופס לעיצוב האלמנט מחולק לארבעה חלקים:
חלק ראשון - בחירת אלמנט
חלק שני - עיצוב האלמנט
חלק שלישי - עיצוב הטקסט עם המשתמש/ת הוסיפ/ה טקסט
חלק רביעי - עיצוב מיוחד שספציפי לאלמנט אותו בחר/ה המשתמש/ת
המשתמש/ת תוכל/יוכל לשמור, למחוק ולטעון את הדף.
האפליקציה היא רספונסיבית
